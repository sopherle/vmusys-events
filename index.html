VMUSYS Monthly Coffee Break

Recommended: download the calendar file and import it into your own calendar.

Download .ics (import into your calendar)

How to import
• Apple Calendar (macOS): File → Import… → select the .ics → choose your calendar
• Outlook: File → Open & Export → Import/Export → Import an iCalendar (.ics)
• Google Calendar: Settings → Import & export → Import → select the .ics → choose your calendar

Prefer automatic updates if details change? Subscribe instead

Note: Import is a one-time copy. If the schedule changes, you’ll need to import again. Subscribing keeps it updated automatically (but shows as a separate “subscribed calendar”).
